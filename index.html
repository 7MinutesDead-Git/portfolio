App: Govee React Dashboard

Fully responsive React+Node app originally created to let my partner control my wifi LED lights from across the globe.

Now I also let interviewers play with it :P

Adjust the sliders and color pickers to change the brightness, color and temperature of the lightbulbs in real time, via REST API and websockets!
You can save currently selected color and brightness as presets as well.

Why use websockets? Because live UI updates look awesome. Ask me about how I used linear interpolation to smooth out the client-side movement of the sliders even during network lag!

App: GimmeDrink

It's like Googling for recipes, without all the bs.

This is a fully responsive app, built purely with raw HTML5, CSS and Typescript (no frameworks needed), for looking up drink recipes by name or ingredient, sourced from multiple databases, and getting back only what you need: A name, a picture, ingredients, and instructions. Node.js

React

I know React so your app looks modern, operates lightning fast on all devices, and has a no-frustration approach to UI and UX design. Your users will visit your site just to enjoy interacting with it.

TypeScript
